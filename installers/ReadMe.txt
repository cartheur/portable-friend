How to setup a Mobile Browser Emulator in Windows Mobile 6:

I assume the user is on a 32 bit system running Windows XP

Download and install Microsoft ActiveSync
Download and install the Microsoft Device Emulator 3
Download and install the Windows Mobile 6 Standard Images: (http://download.microsoft.com/download/0/1/2/012bfbba-9fe5-4e68-86c9-d434446d97dd/0409/Windows%20Mobile%206%20Standard%20Images%20(USA).msi)
Re-Run the Windows Mobile 6 Professional Images installer (Windows Mobile 6 Professional Images (USA).msi) and select the option to Repair
